## Problem Statement
Road accident victims face life-threatening delays due to slow notification systems and difficulty in communicating precise GPS locations to emergency responders.

## Proposed Solution
**RescueLink** is a one-tap web portal that bypasses traditional call centers to send instant, location-verified alerts directly to EMS providers.

## Innovation & Creativity
Our solution uses the browser's native Geolocation API to ensure 100% accuracy in victim location without requiring the user to speak or know their surroundings.

## Technical Complexity & Stack
* **Frontend:** HTML5, Tailwind CSS
* **Logic:** JavaScript (Geolocation API)
* **Version Control:** GitHub

## Usability & Impact
Targeting bystanders and victims. Impact: Reducing "The Golden Hour" response time significantly.

## Setup Instructions
1. Open `index.html` in any modern web browser.
